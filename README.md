# Patrick Heng - Temporary portfolio 2016

#### TECHNOLOGIES

* Preact
* GSAP - TweenMax
* Babel | ES2015
* Webpack
* ESLint
* SASS

##### Install dependancies :
```shell
  npm i
```

##### Launch the project :
```shell
  npm start
```

The project will be launched at http://localhost:3000/

#### Build for production :
```shell
  npm run build
```
